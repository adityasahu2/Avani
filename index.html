What is AVANI?

Avani is a tool designed to help architects and city planners create sustainable communities and developments. It provides data-driven solutions, GIS mapping, and machine learning insights to optimize urban green spaces and support eco-friendly urban planning.

Project Overview

Urbanization is Rapidly Reducing Green Spaces, Leading To Environmental Issues Such As Heat Islands, Pollution, And Biodiversity Loss. City Planners Struggle To Analyze And Optimize Green Areas Effectively. Our Project, AVANI, Provides A Data-Driven Solution Using GIS And Machine Learning To Ensure Sustainable Urban Planning.

The Problem

Loss Of Green Spaces

Rapid urban expansion is reducing tree cover and open spaces.

Lack of Data-Driven Insights

Planners rely on outdated methods without real-time analysis.

Flooding & Soil Erosion

The absence of green spaces affects water absorption, increasing flood risks.

Limited Real-Time Public Awareness

Lack of public access to real-time environmental data limits engagement in sustainable practices.

Inefficient Land Use Planning

There is no structured approach to balancing development with environmental sustainability.

Poor Air Quality & Biodiversity Loss

Less greenery leads to increased pollution and habitat destruction.

Solution

AVANI collects real-time data on urban landscapes from different sub zones of project field using GIS and IoT-based sensors. Machine learning algorithms analyze this data to suggest optimal zones for tree plantations and sustainable construction. The output is a 3D model that helps planners and architects make informed decisions. It improves urban greenery, reduces heat islands, and enhances sustainability. Automated Report Generation: The system generates detailed reports with actionable insights for city planners.

Technologies Used

GIS Mapping

Creating responsive and interactive websites using modern technologies and frameworks for optimal user experience.

IoT-Based Sensors

IoT-based sensors collect real-time environmental data, including temperature, pollution, and soil moisture.

Machine Learning

Machine Learning algorithms process data to generate insights and predictions for urban planning.

3D Vsualization

3D visualization tools help architects and planners design sustainable cities.

Cloud-Based Platform

Cloud-based platform allows stakeholders to access reports and analytics anytime, anywhere.
